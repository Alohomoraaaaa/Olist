# Olist E-Commerce Analytics & Customer Review Prediction

End-to-end Data Analytics and Machine Learning project built using:

- PostgreSQL
- Python
- Pandas
- XGBoost
- Power BI

## Features

- Revenue Analysis
- Delivery Performance Analysis
- Customer Review Analysis
- Predictive Analytics using XGBoost

## Dataset

Olist Brazilian E-Commerce Dataset

https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Model Performance

- ROC-AUC: 0.747
- F1 Score: 0.40
- Recall: 28%
